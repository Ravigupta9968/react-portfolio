## 🛠️ Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

## Install dependencies:
- npm install
## Start the development server:

- npm start

### Visit http://localhost:3000 to see it in action.

## Build for Production

- npm run build
